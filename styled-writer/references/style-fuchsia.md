# Fuchsia Dunlop（扶霞·邓洛普）风格规范

## 选题聚焦

地方饮食文化、普通厨师与食客、传统烹饪技艺、食材流通、地方风俗，以及中国社会变迁如何体现在「吃」之中。核心不是美食评价或推荐，而是通过食物理解地方与人——一道菜为什么在这个地方变成这个样子。

## 分析框架

「饮食细节 + 文化观察」。从一道菜、一次聚餐、一个菜市场或一位厨师的经历切入，逐步展开地方历史、阶层结构、地域性格与时代变化。食物是透镜，不是主角。强调长期浸泡式观察与亲身体验——必须写自己吃过、见过、聊过的东西，避免抽象理论压过真实生活。

## 叙事方式

感官先行。开头常从气味、口感、厨房动作或餐桌场景进入；中间穿插厨师故事、地方掌故、食材来源与个人经历；通过一道菜连接更大的文化背景。叙事强调「人在场」——作者既是品尝者也是学习者，对不熟悉的饮食传统保持谦逊和好奇。

## 语言风格

细腻、克制、准确、充满感官描写。擅长写味道、火候、质地与烹饪动作——「刀落在砧板上的节奏」「油在锅里舒展的姿态」，但不堆砌形容词，不过度炫耀术语。语气带有温和的好奇心与尊重感，像一位长期生活在当地的人在讲述她慢慢学会的东西，而非猎奇的游客。

## 文章结构

```
开头：气味/口感/厨房动作/餐桌场景 → 感官进入
展开：厨师经历 → 食材来源 → 地方掌故 → 个人体验
中段：通过一两道菜连接地方历史、阶层与时代变化
结尾：含蓄余味，不强行升华，像一餐结束后的安静时刻
```

禁止使用小标题。靠场景和感官切换自然推进。

## 输出要求

- 必须包含具体食物细节、人物互动、地方环境与真实感官体验
- 必须体现饮食与社会文化的关联（而非孤立写美食）
- 必须有生活气息与地方性，让读者感觉「闻到了那个地方的味道」
- 避免网红探店感、浮夸赞美（「入口即化」「弹牙」）、空泛乡愁、文化优越感，以及把地方饮食简单符号化为「XX美食的代表」
