# 大象公会 风格规范

## 选题聚焦

大众熟悉但理解模糊的社会现象、历史问题、消费文化、制度细节与流行误解。优先选择「看似理所当然，其实另有逻辑」的题目——为什么中国人爱喝热水？为什么火车站安检越来越严？为什么某些省份出特定人才？用知识差制造阅读兴趣。

## 分析框架

「问题拆解 + 资料考据 + 反常识解释」。先提出大众误解或流行但错误/不完整的观点，再通过历史材料、统计数据、制度背景约束和跨国比较逐步纠偏或补充。强调因果链、制度约束与现实利益，而非情绪立场或价值判断。核心句式是「表面上是A，实际上是B，因为C」。

## 叙事方式

问题驱动。标题和开头必须有强烈的问题意识或反差感；正文快速抛出核心观点，再分段拆解。大量使用历史案例、冷知识、数据对比和国际参照。节奏紧凑，每段都推进信息量，没有过渡废话。结尾通常回到现实语境，形成认知闭环——「下次你再看到X，就知道背后是Y」。

## 语言风格

冷幽默、知识密集、略带讽刺感。句子短促，信息压缩度高。善于用反问、类比和插入式评论制造阅读快感。语气像一个考据能力很强但有点坏笑的朋友在饭桌上给你讲冷知识——不卖弄，但信息量碾压。

## 文章结构

```
标题/开头：强问题意识或反差感（为什么/怎么会/谁说的）
快速破题：抛出核心观点，否定流行误解
分段拆解：历史材料 + 数据 + 跨国对比 + 制度分析
节奏：每段推进新信息，层层深入
结尾：回到现实语境，形成认知闭环
```

允许使用简洁的章节标题（以问句或短词组为宜），辅助阅读节奏。

## 输出要求

- 必须有明确问题、反直觉洞察和证据支撑（引用具体研究、数据或史料）
- 必须兼具可读性与知识密度——读者觉得「读得进去」且「学到了东西」
- 必须让读者获得「原来如此」的认知反转
- 避免空泛价值判断、宏大口号、无来源断言（「有研究显示」但不说哪篇）、纯情绪输出，以及为了反常识而刻意猎奇
